Methods & data source

Source: waffle_houses.csv — a snapshot of Waffle House's public store locator ( numbered U.S. locations, DATA_DICTIONARY.md has the full column docs).

Store number is a proxy for opening order, not exact dates. The data has no opening-date column, so we sort by store number as a stand-in for sequence. This is approximate: numbers are assigned roughly in the order stores open, but there are gaps, retired numbers, and renumbered sites, so a lower number only usually means "opened earlier." What's excluded: record with no numeric store code (the Waffle House Museum, WH_Museum) is left out of the timeline because it has no number to sort by. Coverage is U.S.-only; this is a footprint over store numbers, not a per-capita or chronological-by-year analysis.

Reproducible: the page is generated by build_scrolly.py from the CSV and an offline US-states outline; every dot lands at the latitude/longitude in the data, and all counts in the prose are computed at build time.
